# Frontend - Patient Doctor (Next.js + Tailwind)

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Set environment variables in `.env.local`:
   ```
   BACKEND_URL=http://localhost:4000
   ```

3. Run dev server:
   ```
   npm run dev
   ```

The app provides:
- / (landing + search)
- /search?q=&city= (results)
- /doctors/[id] (doctor profile)
